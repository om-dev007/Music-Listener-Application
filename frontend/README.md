# 🎧 Sportify Frontend

This is the frontend for the Sportify Clone music streaming application.

The frontend will provide the user interface for streaming music and artist features.

## 🚧 Status

Frontend is currently in development.  
UI and pages will be added after backend completion.

## ✨ Planned Features

- Login & signup pages
- Music player UI
- Home & library pages
- Artist dashboard
- Upload interface
- Responsive design

## 📦 Tech Stack

- React.js
- React Router
- CSS / Tailwind (planned)
- API integration with backend

---

This frontend will connect to the Sportify backend API.
