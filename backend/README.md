# 🎧 Sportify Backend API

This is the backend API for the Sportify Clone music streaming application.

It handles authentication, authorization, and music upload functionality for artists.

## 🚧 Status

Backend is mostly complete.  
Some features and improvements are still being added.

## ✨ Features

- JWT authentication
- Role-based access (artist/user)
- Music upload (ImageKit)
- MongoDB database
- Protected routes
- Artist music creation

## 📦 Tech Stack

- Node.js
- Express.js
- MongoDB + Mongoose
- ImageKit (media storage)
- JWT

## 📌 Main Modules

- Auth system
- Music management
- File upload service
- Role authorization
