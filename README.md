# 🎧 Sportify Clone

Sportify Clone is a Spotify-inspired music streaming web application currently under development.

The project will allow users to stream music and artists to upload and manage their tracks.  
Right now, the backend is almost complete and the frontend is in progress.

## 📂 Project Structure

sportify-clone/ <br>
│ <br>
├── backend/   → Node.js + Express API  <br>
├── frontend/  → React application (in progress) 

## 🚧 Project Status

- Backend: Almost complete
- Frontend: Not started / in progress
- Features: Being added

## ✨ Planned Features

- User authentication (JWT)
- Artist role system
- Music upload & storage
- Audio streaming
- Music player UI
- Artist dashboard
- Playlists

## 🛠️ Tech Stack

- MongoDB
- Express.js
- React.js
- Node.js
- ImageKit
- JWT
